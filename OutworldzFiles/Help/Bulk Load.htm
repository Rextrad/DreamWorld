Outworldz Bulk Load Oars

 

Bulk load will bring in over 230 regions, all set up and loaded with OARS. 

Set Smart Start

If you do not set any Smart Start Options, it will automatically power off the region when it is finished loading.

If you want Smart Start to be set on all regions,  which is really good idea unless you have 64 GB of RAM, you should do it before you load all 230+ regions!

Set Limits

You should set the Restart Settings for Parallel, with limits, or Sequential as this will be CPU and disk intensive

 

 

Go to Setup-Settings-Regions and click the Bulk Load button.

You need to enter your name or the name of someone who shall become the owner of the region.

Enter an existing estate name. and where the regions will go. The X and Y will always be set automatically to the upper right corner of your grid.


 

When you click Bulk load, it will create all regions. It will then boot them one at a time and load the OAR.

If this fails, you can start over.  Any time you run this, it will add any new OARS and load them.
